Моделирование и сопоставление

Есть разные способы создания уровня сопоставления и уровня объектов.

  • Сформировать модель из существующей базы данных можно с помощью Entity Framework Tools. При этом создается концептуальная модель по умолчанию и сопоставление, которые можно настроить с помощью конструктора моделей EDM. Концептуальную модель также можно создать графически с помощью конструктора моделей EDM, а затем сформировать базу данных на основе метаданных, построенных этим средством по модели.
  • С помощью метода разработки Code First концептуальную модель можно определить в коде. Entity Framework выводит концептуальную модель, основанную на типах объектов и дополнительных конфигурациях, которые можно задать. Метаданные сопоставления формируются во время выполнения на основе сочетания определений типов домена и дополнительной информации о конфигурации, которая указана в коде. Можно либо сопоставить модель с существующей базой данных, либо сформировать новую базу данных из модели.
